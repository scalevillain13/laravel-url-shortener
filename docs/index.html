Laravel 13 · Filament v3

Сокращатель ссылок

Laravel-приложение для создания коротких ссылок, отслеживания переходов и управления ими через личный кабинет.

Возможности

  • Редирект GET /{code} и асинхронная статистика кликов
  • Filament admin: ссылки, фильтры, UTM, срок жизни
  • REST API на Sanctum, Docker Compose, GitHub Actions CI
Исходники на GitHub ↗

Это backend-приложение на Laravel — для демо запустите локально через php artisan serve или Docker Compose. Инструкции в README репозитория.
